python语法 常用模块

一、datetime模块

from datetime import datetime,date,time
print(datetime.now())         # datetime.now()返回当天的日期和时间
today = datetime.now()        
print(datetime.date(today))   # datetime.date(t)返回t对应的日期,t是datetime类型
print(datetime.time(today))   # datetime.time(t)返回t对应的时间，t是datetime类型
print(datetime.ctime(today))  # datetime.ctime(t)返回t对应的“星期，月，日，时，分，秒，年”格式的字符串
print(datetime.utcnow())      # datetime.utcnow()获取当前的UTC时间，datetime类型.eg.UTC是世界标准时间
print(datetime.timestamp(today)) # datetime.timestamp(t)返回t对应的时间戳
print(datetime.fromtimestamp(datetime.timestamp(today))) # datetime.fromtimestamp(时间戳)，返回值是datetime类型

date1 = date(2020, 6, 6)      # 使用date类，生成date1对象
time1 = time(18, 1, 1)        # 使用time类，生成time1对象
print(date1)
print(time1)
print(datetime.combine(date1, time1))  # 绑定时间和日期，生成一个新的datetime对象

dt = datetime.strptime("2020/6/6 18:08", "%Y/%m/%d %H:%M") # 用字符串和自定的格式生成datetime对象
print(dt)
print(today.isocalendar())    # ISO格式的日期
print(today.strftime("%Y years %m months %d days %I hours %M minutes %S seconds %p"))   # 对datetime格式自定义对象

二、math模块

  • 浮点数取整，用trunc(x)取整，x是浮点数。返回结果是整数。注意：x//1也是取整，但是返回的结果是浮点数。
import math
math.trunc(3.9)

     运行结果：3
     eg. 3.9//1 的运行结果是3.0

  • 取大整数，使用math.ceil(x)，x是浮点数
price = 3.28
math.ceil(price)

     运行结果：4

  • round(x)进行四舍五入，x是浮点数

  • 对元组中的每个数求和math.fsum(元组)

import math
nums = (1.1, 2.2, 3.3)
print(math.fsum(nums))

     运行结果： 6.6
     math.fsum()和sum()的区别是：math.fsum()只返回浮点数，sum()根据参数决定返回值类型，如果参数是整数，返回值是整数。如果参数是浮点数，返回值是浮点数。

  • 求数的绝对值，浮点数使用math.fabs()，整数使用abs()

三、random模块

  • random() : 生成一个基于 [0.0, 1.0) 之间的浮点数
  • uniform(a,b): 在指定的范围获取随机浮点数 N， N在[a, b]范围内
  • triangular(low, high, mode):返回三角分布的随机数， low、high是返回值的上下界，mode为中值。low和high的默认值分别为0和1。mode参数默认为两个边界的中间点。持续使用该方法，可以得到以mode为对称点的随机分布数据集。
  • betavariate(alpha, beta): 求Beta分布的随机数，alpha和beta都要大于0，返回值介于0,1之间。

    补充知识:漫谈三点估算法中关于贝塔分布、三角分布的理解

import random
print(random.random())

print(random.uniform(1,10.5))

print(random.triangular())
print(random.triangular(0, 10))
nums = [random.triangular(0,10,5) for i in range(100000)]
num1 = 0
num2 = 0
for x in nums:
    if x <= 2:
        num1 = num1 + 1
    else:
        num2 = num2 + 1
print("num1 = %d num2 = %d"%(num1,num2))
print("%f%%   %f%%"%(num1/1000, num2/1000))

print(random.betavariate(5, 3))

四、os模块

  • os模块为访问操作系统提供了相关的之处功能，包括Windows,linux,UNIX,Max OS等。

  • os模块的功能包括：

    • 对文件的相关操作功能
    • 系统访问Path路径的操作
    • Shell命令行操作
    • Linux扩展属性操作
    • 流程管理
    • CPU等硬件信息的获取
    • 基于操作系统的真正的随机数的操作
    • 相关的一些系统常量的提供

  • os模块常用的函数

    • environ函数：获取操作系统的环境变量

      import os
os.environ

    • getcwd函数：获取当前路径对应的字符串

      os.getcwd()

    • system(commend)函数:在shell中执行命令（command，是一个命令对应的字符串）

      os.system("ping 127.0.0.0")

    • urandom(n)函数:获取适合用于加密的n字节大小的随机数字符串

      os.urandom(10)

五、sys模块

  • sys模块提供与python解释器紧密相关的一些变量和函数。

  • 常用的函数

    • path 获取模块文件搜索路径的字符串，或者临时指定新的搜索路径

      import sys
sys.path

      import sys
sys.path[0] = 'd:test'

      eg. sys.path是list类型的变量，可以进行暂时的修改，用于临时指定新的搜索路径。

    • sys.platform 操作系统识别符判断函数

      • Linux -> linux

      • Windows -> win32

      • Windows/Cygwin -> cygwin

      • Mac OS -> darwin

      • 函数作用：根据platform和path实现不同版本的模块文件的导入。例如在Windows操作系统下导入Windows版本的模块，在Linux操作系统下导入Linux版本的模块。

          if sys.platform.startswith('win32'):
      print('调用Windows版本的模块文件代码')
  elif sys.platform.startswith('linux'):
      print('调用Linux版本的模块文件代码')

    • sys.getwindowsversion() 获取当前正在运行的Windows版本信息的元组。

      • major -> 主要版本号
      • minor -> 次要版本号
      • build -> 发布的年月
      • platform -> 操作系统产品类型，2应该表示用户PC版本，3表示服务器版本。
      • service_pack -> 表示一个字符串

六、time模块

  • time.sleep(s) 程序进程暂停s秒

  • time.clock() 函数把当前的CPU处理器时间用秒为单位的浮点数进行返回。EG.该方法被舍弃，不提倡使用。

  • time.strftime(str) 函数根据时间格式字符串解释当前的时间

      time.strftime('%H hours %M minutes %S seconds')

  • time.time() 函数返回自纪元年起以秒为单位的浮点数时间，就是时间戳 

      from datetime import datetime
  import time
  print(datetime.timestamp(datetime.now()))
  print(time.time())
