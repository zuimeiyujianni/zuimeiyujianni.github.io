python语法 类的基本语法

一、类的格式

  • 类的第一行格式

  • class 类名():

  • 类文档说明，使用三个单引号

  • 类函数：或者叫做类方法，这种方法必须依赖类或者实例而存在。

  • __init__ 是类的构造函数，例子如下： 

def __init__(self): # 必须加上self
    name = "西瓜"

  • self关键字作用：用于传输示例对象，实际上传递的是实例对象在内存中的一个地址。调用实例的方法时候，self是隐形传递的。

    例子:

class watermelon():
    def __init__(self, attributes):
        self.attributes = attributes
    def disp(self):
        for attribute in self.attributes:
            print(attribute)

watermelon = watermelon(['绿色', '甜的'])
watermelon.disp()

二、类的属性和方法

  • 属性的调用格式：
    实例名.属性名
  • 方法的调用格式：
    实例名.方法名()

三、类的改造（继承、重写方法）

继承

  • 继承的格式1：
    class 子类名(父类名)
  • 继承的格式2：
    class 子类名(父类名1, 父类名2, 父类名3)

例子： 

class watermelon():
    def __init__(self, attributes):
        self.attributes = attributes
    def disp(self):
        for attribute in attributes:
            print(attribute)

class bigwatermelon(watermelon): 
    def __init__(self, attributes, newattribute):
        super().__init__(attributes)
        self.newattribute = newattribute
    def disp(self): # 重写方法
        for attribute in self.attributes:
            print(attribute)
        print(self.newattribute)

bigwatermelon = bigwatermelon(['绿色', '甜的'], '大的')
bigwatermelon.disp()

重写方法

  • 要求：子类的方法名称和父类方法名称相同

四、私有

  • 让类定义的 函数 or 变量 变成 私有 ，只需要在它的名称前面加上双下划线即可。

五、把类放到模块当中

  • 将类的信息保存到xxx.py文件中。使用下面方法导入该xxx.py文件中的所有类：
    from xxx import * # 从xxx模块中导入所有类
    import xxx as yyy # 导入xxx模块并简称为yyy
    from xxx import clazz # 从xxx模块中导入clazz这个类
