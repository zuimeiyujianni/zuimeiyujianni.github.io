一、我的考研概况

初试成绩 政治 英语一 数学一 专业课845 初试排名 复试成绩 总成绩 总排名
328 64 74 86 104 65/84 167 495 50/70
  • 因为专业课成绩在2020年考研中还可以，所以我这里主要讲一下 845专业课 的复习，希望给以后打算考南大计算机的同学做一个参考。另外，每个人的方法都是不同的，合适自己的才是最好的。

二、专业课复习

845专业课目录

  • 计算机系统基础
  • 数据结构
  • 计算机网络
  • 算法
  • 操作系统

计算机系统基础：

推荐的书籍： 《计算机系统基础》，《计算机系统基础-习题解答与教学指导》

推荐网课：MOOC四门网课《计算机系统基础（一）》，《计算机系统基础（二）》，《计算机系统基础（三）》，《计算机系统基础（四）》

建议：

  • 复习的时候不建议直接阅读《计算机系统基础》，因为《计算机系统基础》书籍中很多内容并不会进行考查，而且内容难度相比其他科目较难。建议直接看MOOC上袁春凤的课程，在每一章结束之后做《计算机系统基础-习题解答与教学指导》上对应的习题。

  • 重点复习章节：

    • 第二章节–数据的机器级表示和处理

      容易考查选择题，另外这个章节是第三章节的基础。

    • 第三章节–程序的转换与机器级表示

      重要!南京大学的845科目自从添加了计算机系统基础课程后每年都从这个地方出大题目。因为在做完相应的章节练习之后，最好能够独立完成”二进制炸弹“(IA-32)，因为它包含了字符串、循环、条件/分支、递归调用和栈、指针、链表/指针/结构各个题型的内容。这是题解

    • 第六章节–层次结构存储系统

      在845没有加入计算机系统基础课程之前，曾经考察过层次存储系统的大题目。另外该章节的内容容易在其他的题目中进行简单的考察。

  • 不建议使用《深入理解计算机系统》进行复习，因为这本书采用的是 X86 体系架构，而《计算机系统基础》采用的是 IA-32。而且《深入理解计算机系统》内容较多，建议 考研之前 进行阅读or 考研之后 进行阅读。此外，《计算机系统基础》参照《深入理解计算机系统》进行编写。

  • 之前没有上过《计算机系统基础》课程的同学需要留出 足够的时间 复习计算系统基础

  • 加入相应的群，遇到不会的问题问一下。21届的学生可以加入“2021南大CS考研劝退”群（qq群号：550150534），里面有很多已经上岸的学长学姐和正在考研的同学。里面经常会交流一些专业课的内容。

数据结构 :

推荐的书籍： 《王道 数据结构》，《数据结构 殷人昆》

建议：

  • 先做王道的习题。《数据结构 殷人昆》这本书可以不看，做完王道之后需要看看《数据结构 殷人昆》上的经典例子，例如：约瑟夫环。之前有考查书上经典例子。
  • 王道的数据结构缺少的内容包括
    • P/NP/NPC/NP-Hard问题（南大考查一道选择题）
    • 时间复杂度的符号（包括O在内的一共五个，考查选择题，大题会考查时间复杂度）
    • 广义表
    • 并查集(王道讲的很少)，图(王道关于图的算法讲的很少)
    • …………
  • 数据结构的内容比较零碎，建议做两遍王道数据结构，第一遍以理解为主，第二遍是以记忆为主（在冲刺阶段开始，如果基础感觉第一遍做的雨里雾里的同学可以早一点开始）。可以把一些比较难记的内容背下来，比如：B树和B+树各有什么特点、哪些排序算法是稳定的等等、深度优先搜索和宽度优先搜索的模板。
  • 建议时间充裕的同学将王道上的编程题上机运行一下，因为很多问题如果不在机器上面运行很难发现。

算法 :

  • 官网推荐的书籍是黄宇的书。首先同学需要买一本看看。我当时因为一些原因没有买过也没看过这本书，但是一起考研的群友挺推荐这本书的。
  • 对于有过算法经验并且时间充裕的同学，建议阅读《信息竞赛一本通》中的 图， 动态规划 以及, 另外建议阅读《算法笔记》中的 图 部分内容。南京大学的算法题目前主要是 图 为主要考察内容， 2019年考了一次 动态规划 的经典模板题型。 我今年考场考的是 图 的搜索，复试的 C++ 科目考察了 动态规划 的经典模板题 编辑距离（20分）。此外，选导师的时候编程考查了 树的遍历 和 贪心算法。所以，想要考南大的学生要多多练习练习自己的算法能力。大二的学生可以多多参加一些算法的练习，不仅仅是为了考研，找工作的时候，基本的算法能力是基础（参加招聘的时候，往往需要首先进行笔试测试，笔试测试通过之后才可以进入面试。）。

操作系统 :

  • 推荐书籍：《天勤 操作系统》。不建议使用王道的操作系统，主要是因为感觉王道的操作系统有很多东西没有讲清楚。(因为我首先买了王道的操作系统，然后重新买了天勤的操作系统)
  • 推荐课程：腾讯课堂的《天勤率辉考研操作系统》，我当时趁着优惠花了几百元买的课程。感觉非常适合我，因为讲课的速度很慢并且知识点非常的清晰，刚好我可以接受。建议同学先去试听一下，感觉合适再购买课程。
  • 对于操作系统，845主要考察的就是 PV操作 等经典的题型， 特别是PV操作。这里的PV经典问题要熟记到脑海中有对应的代码并且知道每一句代码的意思。以 读写者问题 为例，熟记 读者优先，写者优先， 读写公平 三种实现的方法， 知道如何从 读者优先 转化为 写者优先 or 读写公平， 能够默写相关的操作代码。
  • 补充推荐书籍 PV 金典 。这本书上面有很多的PV操作题目（虽然这本书错误内容有点多），但是要多做做各种PV题才能真正的掌握，这样遇到变形的题目才可以游刃有余。
  • 操作系统部分的知识很多与计算机系统基础上面的范围重叠且不一致，这里不一致的内容一律采用计算机系统基础的说法。

计算机网络：

  • 推荐书籍：我使用的是《天勤 计算机网络》，但是效果并不是很好。因为到了后期刷真题的时候发现大题目都不会做（主要是没有学过对应的知识点），估计是因为没有使用参考书籍的原因。（《自顶向下计算机机网络》这本书是推荐的教材，大家最好买一本，我之前没有买这本书，等我想买的时候，已经临近考试了。）
  • 2020的考试计算机网络的大题相对比往年要容易了一些，因为今年考查的内容是我们考研复习书籍包括的内容（845最恶心的操作就是没有考试范围，不知道从哪里下手。今年手下留情了）。不知道能够成为以后的趋势。
  • 推荐材料：南京大学软件学院的计算机网络PPT，可以下载浏览浏览。

三、其他方面的建议

  • 考研复习的时候最好有一个 稳定且安静 的环境。之前我是在致明楼316复习的，那里的环境非常的安静，学习氛围非常好。这里加上张图片。

    后来，暑假的时候我转移到了竞秀北楼2楼复习，这里也非常好，就是太热了。再贴上一张图片。

    最后，就是最痛苦的图书馆学习生活，每天抢座位让人头秃。

  • 最好有一起考同一所学校小伙伴一起学习，这样可以相互监督一下。

  • 845专业课复习比较慢且不容易拿到高分，所以不要太在意别人专业课复习的 进度 。

  • 确定了学校之后，不要不断的降低自己的目标 ，适当的时候可以询问老师的意见。另外，觉得学的不错的同学可以适当的提高自己的目标。

  • 四六级 要通过，因为四六级也是后面复试要使用的材料。( 特别担心后面英语复试的同学 )

四、学长寄语

    希望大家考研和出国的都考上心仪的大学、就业的都找到自己心仪的工作。另外，大家考研最后一定要 坚持下去 ，"研路慢慢，唯有坚持"。相信我们信息工程学院的学子都越来越好。

如果没有目标，就不会被任何人期待。 —-《垫底辣妹》
