二、字体

  • 加粗
    使用两个星号进行左右包围
  • 斜体
    使用一个星号进行左右包围
  • 斜体加粗
    使用三个星号进行左右包围
  • 删除线
    使用两个波浪线进行左右包围
**加粗**
*斜体*
***斜体加粗***

三、引用

欢迎来南京！
欢迎来南京市！
南京欢迎您!

>欢迎来南京！  
欢迎来南京市！   
南京欢迎您!

引用可以进行嵌套：

欢迎来南京！

南京欢迎您!
南京欢迎你！

>欢迎来南京！  
>>南京欢迎您!  
南京欢迎你！

五、图片

![图片alt](图片地址 "图片title")
alt是图片下面的文字，是对图片的解释。图片title是图片的标题，当鼠标移动到图片上方时候显示的内容。title可以省略

上传本地图片直接点击导航栏的图片标识，然后选择图片。

六、超链接

语法：

[超链接名称](超链接地址 "超链接title")
title是可选的内容

例如：
百度官网地址
百度官网地址 

[百度官网地址](https://www.baidu.com/)  
[百度官网地址](https://www.nju.edu.cn/ "这是百度的官网哦")

注意：不支持链接在新的页面打开，简书做了处理是可以的。如果想要打开新的页面使用下面的语法： 

<a href="超链接的地址" target="_blank">超链接名称</a>  
<a href="www.baidu.com/" target="_blank">百度官网</a>

百度官网

七、列表

无序列表

语法：
使用+ - *任何一种都是可以的：

- 狗子1号
+ 狗子2号
* 狗子3号
  • 狗子1号
  • 狗子2号
  • 狗子3号

    有序列表

    语法：
    数字加点 
1. 狗子一号
2. 狗子二号
3. 狗子三号
  1. 狗子一号
  2. 狗子二号
  3. 狗子三号

    列表的嵌套

    语法：
    上一级和下一级之间敲三个空格即可
  • 这是第一级
    • 这是第二级
      • 这是第三级
- 这是第一级
   - 这是第二级
      - 这是第三级
  • 这是第一级第一个元素
  • 这是第一级第二个元素
    1. 这是第二级第一个元素
    2. 这是第二级第二个元素
    3. 这是第二级第三个元素
      • 这是第三级第一个元素
      • 这是第三季第二个元素
* 这是第一级第一个元素
* 这是第一级第二个元素
   1. 这是第二级第一个元素
   2. 这是第二级第二个元素
   3. 这是第二级第三个元素
      * 这是第三级第一个元素
      * 这是第三级第二个元素

八、表格

语法：

表头|表头|表头
---|:--:|--:
内容|内容|内容|
内容|内容|内容|

示例：
姓名|年龄|性别
—|:–:|–:
狗子|100|女|
二哈|100|女|

九、代码

语法：
当行代码：代码使用`左右包起来

`int main(void)`

int main(void)
多行代码：使用三个反引号左右包起来

#include<stdio.h>
int main(void){
    printf("Hello world");
}
